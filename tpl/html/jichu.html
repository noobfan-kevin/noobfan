页面的构成

html页面由两大部门构成---页面头和页面体。

看下面的例子：

<html>
<head>
</head> //
<body>
<h2>noobfan 是学习编程的好地方。</h2>
<p>我的 noobfan</p>
</body> //
</html> //

两个head之间的部分称之为头部分。

两个body之间的部分称之为体部分。

所有元素都应该被html标签包裹。

页面的头部分

通过上部分我们得知，位于两个head之间的属于页面的头部分。

头部分的主要内容为：

1.页面信息。页面的名称，以及一些属性信息，这些信息可以方便网页爬虫爬到该网页信息。

2.第三方的文件引入。通过<link>标签引入样式，通过<script>标签引入动作。

看下面的例子：

<html>
<head>
<link href="css/noobfan.css" rel="stylesheet"/>
<script src="js/noobfan.min.js"><script />
<head/> //
<body>
<h2>noobfan 是学习编程的好地方。</h2>
<p>这是我的noobfan</p>
</body> //
</html> //

页面的体部分

通过上面部分我们得知，位于两个body之间的属于体部分。

体部分是页面的呈现部分，页面的显示内容都是写在体部分中的。

看下面的例子：

<html>
<head>
<link href="css/noobfan.css" rel="stylesheet"/>
<script src="js/noobfan.min.js"><script />
<head/> //
<body>
<h2>noobfan 是学习编程的好地方。</h2>
<p>这是我的noobfan</p>
</body> //
</html> //
